# The Silver Tower

A responsive web application prototype inspired by tabletop RPG reference tools.  
This project was developed as a Minimum Viable Product (MVP) for a web development course.

---

## 📖 Project Overview

**The Silver Tower** is a front-end web application that demonstrates:

- Responsive layout design
- Dynamic content rendering using JavaScript
- Data separation using external JSON
- Modular project structure

The goal of this MVP is to simulate a scalable architecture similar to real-world reference platforms.

---

## 🛠 Technologies Used

- HTML5 (Semantic Structure)
- CSS3 (Responsive Design, Grid, Variables)
- JavaScript (DOM Manipulation, Fetch API)
- JSON (External Data Source)
